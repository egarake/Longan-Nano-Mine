Longan Nano Mine Sweeper Game

[Hardware]

VR Left GND A0 3.3V

VR Right GND A0 3.3V

SW BOOT0 A8 3.3V

[How to play]

Game Start BOOT0 SW

Mode Select Right VR

Select row by right volume.

Select column by left volume.

Open with BOOT0 SW.
